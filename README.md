# Global Sound & Stem Archive (GSSA)
**Information Science 4730: Digital Curation Final Project**

## Project Overview
The GSSA is a curated repository designed to preserve the high-fidelity building blocks (stems) of modern African music. By archiving individual stems in lossless formats, we ensure that the cultural and technical DNA of these tracks remains accessible for future remixes, research, and preservation.

## Navigation & Use
- **Search:** Users can search the `/metadata` directory to find stems by BPM, Key, or Language.
- **Access:** High-fidelity WAV files are located in the `/items` directory.
- **Requirements:** These files are optimized for use in any modern DAW (Logic Pro, Ableton, FL Studio).

## Preservation Strategy
- **Format:** All items are stored as 24-bit WAV files to prevent data loss.
- **Documentation:** Technical metadata (BPM/Key) is embedded in the `metadata.csv` to ensure interoperability.
